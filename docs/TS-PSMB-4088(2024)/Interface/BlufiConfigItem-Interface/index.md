# 概述

本文档详细说明了基于 Blufi 协议的设备配置接口。通过该接口，可以实现：

1. 设备基础信息的读写
2. 网络连接参数的配置
3. MQTT服务器连接参数的设置
4. 设备运行参数的调整
5. 设备状态的监控和诊断

所有配置项均采用 JSON 格式进行数据交换，确保了数据传输的可靠性和可读性。
